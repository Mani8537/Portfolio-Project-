# 🚀 Alex Morgan – Developer Portfolio

A clean, responsive personal portfolio website built with **HTML, CSS, and Vanilla JavaScript**. No frameworks, no build tools, no dependencies — just open the browser and it works.

## ✨ Features

- Fully responsive (mobile, tablet, desktop)
- Smooth scroll navigation
- Animated skill bars
- Scroll-reveal animations (Intersection Observer API)
- Contact form with feedback
- Dark theme with modern UI
- Zero npm dependencies — pure HTML/CSS/JS
- Ready to deploy on GitHub Pages in 2 minutes

## 📁 Project Structure

```
portfolio-project/
├── index.html          # Main HTML file
├── css/
│   └── style.css       # All styles
├── js/
│   └── main.js         # Interactions & animations
└── README.md           # This file
```

## 🛠️ How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/portfolio-project.git
   ```
2. Open the folder:
   ```bash
   cd portfolio-project
   ```
3. Open `index.html` in your browser — that's it!

## 🌐 Deploy on GitHub Pages

See the full step-by-step deployment guide below in this README or follow the [GitHub Pages documentation](https://pages.github.com/).

**Quick Steps:**
1. Push this project to a public GitHub repository
2. Go to **Settings → Pages**
3. Set source branch to `main` and folder to `/ (root)`
4. Click **Save** — your site is live at `https://YOUR_USERNAME.github.io/portfolio-project`

## 🎨 Customization

Edit `index.html` to update:
- Your name, role, and bio
- Skills and percentages
- Project names, descriptions, and links
- Contact information and social links
